# CSS selector

## *
전체 선택자

## tag
지정 선택자

## tag, tag..
복수 지정 선택자

## #id
id 선택자, #id명, id는 페이지내 한번만, id는 최소한사용

## .class
class 선택자, .class명, 비슷한 스타일끼리 묶어서 구조화할 수 있다.

## Descendant selector
자손 선택자, parent-tag child-tag(anchor tag), 부모태그 내 자식태그를 선택, class 사용가능
